# Push-swap

## Description
Push-swap est un programme qui trie une pile d'entiers en utilisant un ensemble limité d'opérations. Il implémente l'algorithme de tri appelé "RADIX".

## Installation
1. Clonez ce dépôt sur votre machine locale.
2. Assurez-vous d'avoir le compilateur C installé.
3. Exécutez la commande `make` pour compiler le programme.

## Utilisation
1. Exécutez le programme en utilisant la commande `./push_swap [liste d'entiers]`.
2. Par exemple, pour trier les nombres 5, 2, 8, 1, 3, vous pouvez exécuter la commande `./push_swap 5 2 8 1 3`.
3. Le programme affichera la liste des opérations nécessaires pour trier les nombres.

## Exemples
- Pour trier les nombres 4, 1, 7, 3, 2 : `./push_swap 4 1 7 3 2`
- Pour trier les nombres 10, 5, 8, 2, 6 : `./push_swap 10 5 8 2 6`
